## Vue d'ensemble

Outil de génération d'étiquettes pour l'épicerie coopérative rennaise Breizhicoop, via l'utilisation d'APIs (ERP Odoo, OpenFoodFact).

## Technologies utilisées

- **Backend**: Python, Django
- **Frontend**: jQuery
- **Styling**: Bulma
- **APIs**: ERP Odoo, OpenFoodFact
